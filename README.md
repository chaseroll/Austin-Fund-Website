# Austin Fund

Website for The Austin Fund — pre-seed and seed stage venture capital backing University of Austin affiliated founders.

## Color Palette

| Name        | Hex       | Usage                                              |
|-------------|-----------|-----------------------------------------------------|
| White       | `#EAEAE9` | Light backgrounds, text on dark                     |
| Accent      | `#1A1C13` | Italic "Fund", section labels, secondary text       |
| Black/Green | `#0D0E0A` | Dark backgrounds, primary body text                 |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js (App Router)
- Tailwind CSS
- Framer Motion
- General Sans font
